# Open Graph Images

This directory should contain Open Graph images for social media sharing.

## Required Images:
- `og-default.jpg` - Default OG image (1200x630)
- `og-fund.jpg` - Fund page OG image (1200x630)
- `og-chat.jpg` - Chat page OG image (1200x630)
- `og-signup.jpg` - Signup page OG image (1200x630)

## Specifications:
- **Size**: 1200x630 pixels
- **Format**: JPG or PNG
- **File size**: Under 1MB
- **Design**: High contrast text, clear branding

Once created, update the metadata in each page/layout to reference these images.
